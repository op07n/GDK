Detailed Description

Represents a Window within a desktop windowing system.

Window is responsible for initalizing the graphics context and cleaning it up. GLEW, GLFW libs are initialized at construction time of the first window. They are cleaned up in the destructor of the last window.

call update whenever you want to process the window's messages. call draw whenver you want to update the window's graphics.
