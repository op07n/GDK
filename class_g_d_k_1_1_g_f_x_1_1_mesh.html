Member Enumeration Documentation

enum GDK::GFX::Mesh::Type
strong

Hint to the graphics device about how the vertex data will be used. Generally, dynamic data (data that is likely to be frequently rewritten) will be placed in video memory with fast read write speeds while static will be placed in slower (and more plentiful) video memory. Exact behaviours are implementation specific.
