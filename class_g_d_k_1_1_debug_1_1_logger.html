Detailed Description

Used to render debug messages in some form. The default behaviour is to send the data to std::clog, however this can be changed by passing a function of sig void(const std::string&) to the constructor. In this way, Logger can be used to output to files, pipes, over the network, etc.

This header also declares the free standing functions Debug::log and Debug::error, which are GDK wrappers for std::clog and std::cerr respectively.
